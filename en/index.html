Nepal Weather API

नेपाली

About

This is the API for weather in Nepal. The data provided in this api is scrapped from http://mfd.gov.np/weather/. Its source code is available in the github. To contribute or go to source code Click here

USING THE API

This api uses get request and returns value in json format.
pass ?place=placenamefor weather in English
pass ?placenp=placename for weather in Nepali

eg. https://nepal-weather-api.herokuapp.com/api/?place=all will return something like
